# /components

Reusable UI building blocks, split by domain: `ui/` (buttons, inputs, modal,
card, badge, etc.), `product/`, `cart/`, `checkout/`, `admin/`, `layout/`.

Populated in **Phase 4** (ui, product, layout) and **Phase 5** (cart,
checkout, admin). Defined in `docs/11-reusable-components.md`.
